Hacer semántico

4. Mostrar / Ocultar elementos

¿A quién se lo queres ocultar?

  • Al usuario "común"
  • Al usuario de SR

Ocultar al usuario "común"


.sr-only{
	border: 0;
	clip: rect(0 0 0 0);
	margin: -1px;
	overflow: hidden;
	padding: 0;
	position: absolute;
	width: 1px;
	height: 1px;
}

Ocultar al usuario de SR


<el aria-hidden="true">...</el>

Visual
